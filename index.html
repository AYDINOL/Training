Intro to Web Development

HTML (HyperText Markup Language)

HTML is a markup language which you use to write and structure your content. You can create websites using HTML. You can structure your website content further by using tags. These tags allow you to add paragraphs, use bulleted lists, align the text and more. HTML runs on the browser. It doesnâ€™t tell the computer to calculate so it's not a programming language.

CSS (Cascading Style Sheets)

CSS is a style sheet language which you use to style your HTML elements. With CSS, you can define a background color, change the font or font size, add an image and more. CSS runs on the browser so it's not a programming language.

JavaScript

JavaScript is a programming language which you can use to change your website into a more dynamic, more interactive one. With JavaScript, you can add features like drop down menus, search suggestions, animations and more. It can be written alongside HTML.

PHP(Hypertext Preprocessor)

PHP is a general purpose scripting language. You can create dynamic and interactive websites. Unlike HTML, CSS and JavaScript, PHP runs on a server. Yet, it can be embedded into HTML.

Python

Python is a general purpose programming language. Unlike the topics above, Python is a backend development tool. You can use it to write algorithms. These algorithms can solve problems or create new products. You can use Python to create websites too but it's more used to create softwares.

Programming

Programming is the act of writing computer programs. When you program, you tell a computer how to do a certain task.

Algorithm

Algorithms are a set of rules or methods that computers follow to solve a problem.

Web Server

Web server is a software in a computer which collects webpage materials in one place. When you search for a website on the internet, your request goes to that server. As a response from the web server, you can reach the website on your browser.

Live Server

Live server is a server which you can reload, edit in real time. It's a more dynamic access to the browser content.

Auto-complete

Auto-complete is a feature which completes your words while you are typing them.

Tags

Tags are keywords that tell the browser how to display the content.

GitHub

GItHub is a platform where you can write, update and share codes. You can work on creating softwares on GitHub with your friends.

IDE (Integrated Development Environment)

IDE is a software in which you can use to write codes.
